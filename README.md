# Project description
This project develops a simple speech assistant that helps students and applicants with the admission and application process at THD.
It answers questions about application requirements, necessary documents, submission deadlines, and contact information.
The assistant will be implemented in **Python**, using **keyword spotting** for intent detection and a **Streamlit** interface for user interaction.

# Installation
pip install -r requirements.txt

Used versions:
- Python 3.12.x
- Streamlit 1.36.x


# Basic Usage
streamlit run app/app.py
